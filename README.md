# Movie Catalog (Flutter)

A Flutter project to show list of my favorite movies.

## Running on Local

- Build/get dependencies
    ```bash
    flutter pub get
    ```
- Run project on local (using mobile emulator or web browser)
    ```bash
    flutter run lib/main.dart
    ```


## Platform

- Android
- iOS
- Web
